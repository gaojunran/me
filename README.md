## ✨ Features

### 📊 WakaTime Integration

集成了 WakaTime 编码统计，使用 Netlify Functions 作为代理层，展示：
- GitHub 风格的编码活动热力图
- 编程语言、项目、编辑器统计
- 每日/每周/每月编码时间

**快速开始**: 查看 [WAKATIME_QUICKSTART.md](./WAKATIME_QUICKSTART.md)

**完整文档**: 查看 [WAKATIME_DEPLOYMENT.md](./WAKATIME_DEPLOYMENT.md)
